JNoexs 实战，旷野之息英杰技能无CD
2024-01-14 15:25:31 #Switch

以塞尔达传说：旷野之息为例。游戏信息：

查找地址

英杰技能在冷却时有倒计时，第一直觉是搜索 float 类型，以秒为单位。
在尝试了几次后放弃了，因为它没有按秒去存储。最终搜索模糊值并不断递减搜索，终于找到了 heap 地址：0x1019E0F77C

数据类型猜对了，是 float 类型，但值并不是像我想的那样是按秒去存储的。至于值具体如何计算的不用关心，总之这个值为 0 的时候英杰技能就重置了。

观察偏移量

在游戏暂停的情况下，用 Breeze 搜索剩余时间，将结果添加到书签，再对这个地址设置一个写入断点，目的是为了观察下偏移量，方便后面确认指针的偏移量

代码位置：main + 0x00F3CAAC
指针偏移：0x1E0C

打开 IDA 观察一下

观察前后代码发现有四个 if 代码几乎一样，测试发现这几个地址英杰技能CD时间！

搜索指针

回到 JNoexs 搜索指针，设置如下

  • 深度从1开始查，能查到最好，查不到再加大。
  • 偏移量也是一样，填写从前面断点捕捉到的值，这比软件默认值 F000 少很多，运气好的话可以秒出结果。
  • 过滤看情况设置，先限制在 main 区搜，搜不到再取消过滤。


运气不错，一次就找到了唯一的一个指针。
结合在 IDA 中的分析，确认四个英杰技能 CD 时间的地址非常近：

总结

  • 搜索数据地址需要运气与经验。
  • 对数据地址设置写入断点，观察指针偏移，这有助于确认正确的指针，也能在搜索指针时节省大量时间。
  • 搜索指针时秉着从小到大的范围搜索，不要一上来就把搜索范围弄的很大，这样很容易搜索海量指针。
