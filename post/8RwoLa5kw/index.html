[Sciter] 样式集

2021-07-30

在Sciter中有个叫样式集的概念，它主要的目的是解决程序应用主题样式的问题。

@set my-btn {
    :root {
        background: red;
    }
}

button {
    style-set: my-btn;
}

样式集使用@set关键字指定一个名称。:root是伪元素，表示应用该样式集的元素。
应用样式集用style-set关键字。清除样式集可以用style-set: none。

优先级

样式集的权重比普通CSS样式高

<html>
<head>
    <style>
        @set my-btn {
            :root {
                background: red;
            }
        }

        button {
            style-set: my-btn;
        }

        #test {
            background: green;
        }
    </style>
</head>
<body>
    <button id="test">test</button>
</body>
</html>

元素背景色将是红色，也就是说元素自身的样式没有效果。如果需要覆盖样式集的属性可以用!important关键字。

#test {
    background: green !important;
}

继承性

样式集是可以继承的

@set my-btn-base {
    :root {
        border: 1px solid blue;
    }
}

@set my-btn < my-btn-base {
     :root {
        background: red;
     }
}

使用<表示继承，比如上面的my-btn将从my-btn-base继承所有样式。

内置样式集

Sciter内置了两个样式文件master-base.css和ux-master.css。
可查看samples.sciter\editor-plaintext\sciter-resources.htm。
或使用fetch来获取内置文件

<html>
<head>
</head>
<script>
    async function foo(url) {
        const r = await fetch(url);
        if (r.ok) {
            document.querySelector('plaintext').value = r.text();
        }
    }

    document.ready = () => {
        document.getElementById('f1').onclick = () => {
            foo('sciter:master-base.css');
        };

        document.getElementById('f2').onclick = () => {
            foo('sciter:ux-master.css');
        };
    }
</script>

<body>
    <button id="f1">master-base.css</button>
    <button id="f2">ux-master.css</button>
    <plaintext type="text/css" readonly />
</body>
</html>
