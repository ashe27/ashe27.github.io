去掉POPUP的阴影

Sciter uses two types of popup windows:

1.standard OS popup window, with or without shadow – per user’s / OS preferences
2.“layered” popup window, [semi-]transparent element – without shadow.

Element is considered as transparent if its shape has transparent elements or it has round borders. So either one of these three properties:

popup {
  border: 1px solid transparent; // or
  border-radius: 3dip; // or
  background: transparent url(...); // transparent back and image with alpha
}

will cause layered window to be created and so no shadow.
