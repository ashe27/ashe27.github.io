一些软件最后支持 XP 的版本
2025-07-01 20:55:10 #工具软件

Windows Installer

因为 Windows XP 自带的版本是3.1，所以有可能无法运行某些高版本的.msi文件，建议升级到v4.5版本。
WindowsXP-KB942288-v3-x86.exe

Visual Assist

最高支持到Visual Studio 2017 RC
https://downloadfiles.idera.com/WholeTomato/VA_X_Setup2118.exe

Git for Windows

Git-2.10.2-32-bit.exe
https://github.com/git-for-windows/git/releases/tag/v2.10.1.windows.1

TortoiseGit

安装包：TortoiseGit-1.8.16.0-32bit.msi
简体中文语言文件：TortoiseGit-LanguagePack-1.8.16.0-32bit-zh_CN.msi
https://download.tortoisegit.org/tgit/1.8.16.0/

WinDbg

最后一个能在 XP 上运行的版本是 6.2.9200.16384。XP 系统上安装需要先升级 Windows Installer 版本。
X86 Debuggers And Tools-x86_en-us.msi
X64 Debuggers And Tools-x64_en-us.msi

Synergy

在多台电脑之间共享键鼠的软件。
激活码：7B76313B70726F3BE5BDAC20E585B33B313B73796E313333323040676D61696C2E636F6D3B3B3B7D
synergy-v1.8.8-stable-Windows-x86.msi
https://www.brahma.world/synergy-stable-builds/
https://github.com/amankhoza/synergy-binaries?tab=readme-ov-file#latest-stable-release-v188-stable

TcpView

网络端口查看工具。v3.05
https://www.pc6.com/softview/SoftView_47901.html
https://download.csdn.net/download/lhp121154867/5015730
https://soft.3dmgame.com/down/203592.html
