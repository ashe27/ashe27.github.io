一些开发工具兼容XP的版本
2023-06-30 17:17:56 #工具软件

Windows Installer

因为 Windows XP 自带的版本是3.1，所以有可能无法运行某些高版本的.msi文件，建议升级到v4.5版本。
https://archive.org/download/windows-installer-4.5.7z/Windows%20Installer%204.5.7z
或者搜索引擎找WindowsXP-KB942288-v3-x86.exe这个文件。

Visual Assist

最高支持到Visual Studio 2017 RC
https://downloadfiles.idera.com/WholeTomato/VA_X_Setup2118.exe

Git for Window

https://github.com/git-for-windows/git/releases/tag/v2.10.1.windows.1
直连地址：https://github.com/git-for-windows/git/releases/download/v2.10.2.windows.1/Git-2.10.2-32-bit.exe

TortoiseGit

https://download.tortoisegit.org/tgit/1.8.16.0/
直连地址：https://download.tortoisegit.org/tgit/1.8.16.0/TortoiseGit-1.8.16.0-32bit.msi

简体中文语言文件：https://download.tortoisegit.org/tgit/1.8.16.0/TortoiseGit-LanguagePack-1.8.16.0-32bit-zh_CN.msi
